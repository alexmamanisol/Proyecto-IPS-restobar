##Instruciones

###1. Creen sus apps en este directorio:
  Cada aplicación debe de tener su propio front y backend, para que 
  el codigo sea reutilizable, pueden usar los componentes UI de la carpeta
  ./packages que gestiona Turborepo

###2. Recomendable crear una Branch con asunto de su app:
  Es mejor gestionar las aplicaciones así porque cada actualización 
  que se venga al "master" puede ser conflictiva y tambien muy trabajosa
  de arreglar dependencias, luego realizamos u Merge adecuado a travez de los Github Workflows
